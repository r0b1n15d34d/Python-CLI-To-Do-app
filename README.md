# Python CLI based To-Do App
A simple Python App for managing tasks

## Features
- Adding tasks
- Viewing the current tasks
- Marking tasks as done
- Deleting tasks
- Automatically saves tasks to an external .txt file

## How to use
- Make sure you have Python installed on your computer
- Download the script, open the cmd and run:
```
python to_do_list.py
```
- you can also clone the repository using git:
```
git clone https://github.com/r0b1n15d34d/Python-CLI-To-Do-app.git
```
